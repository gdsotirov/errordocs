Request-URI is longer than the server limit

The server is refusing to service the request, because the Request- URI is longer than the server is willing or able to interpret. This is likely to occur when:

  • your client improperly converted POST request to a GET request;
  • your client has descended into a URI "black hole" of redirection;
  • your may trying to abuse this server.

You may:

  • try to set your agent properly or upgrade it;
  • go away!
